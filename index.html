Heya Stranger!

You might have stumbled on this page wondering what it does. In short, this website allows people to generate fullstack websites by just filling out a small form. Your website will be built using Node.js and will be scalable with any database solution you want. If you don't know what some of those words mean, you probably aren't a fullstack engineer, but that doesn't matter- we do all the heavy lifting for you.

In layman's terms, you can use Squarespace or Wix to build portfolio websites, but not much more. But, you can use Caramocha to generate a website like Twitter or Facebook. Caramocha can even work as a template for more specified websites- instead of rewriting code that is in most apps, you can just generate the 'normal' code with us, and then edit what you want.

So what are you still doing reading? Get to building your dreams!

Some Metadata...

You are going to need to fill out some basic information about what your website is called, what it does, and other things. It isn't completely necessary in production, but it is best practice.

The license is only important if you want to make your website open source, but if not, just choose 'unlicensed'. If you aren't sure which one to pick, just look it up on Google.
